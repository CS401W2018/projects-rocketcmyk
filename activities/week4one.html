1. Explain the difference between an absolute link and a relative link.

- An absolute link uses the complete url which includes the protocol, domain name, and and the path to the resource Meanwhile, a relative link is used to link other pages that are within your website.

2. List the 3 rules with relative links.

Rule 1: Same Folder -file.html

Rule 2: Child folder (nephew) - folder/.file.html

Rule 3: Up (aunt/uncle) "../file.html

3. Do links always need to be inside a NAV tag?

- Links do not always have to be in the nav tag. The navigation usually includes group of links that helps users navigate through different sections of the website. Other links within the page can also be used within tags like the paragraph tag.
