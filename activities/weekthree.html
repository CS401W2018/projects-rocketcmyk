Questions

1. What goes in the header?

-Website Logo and name

2. What goes in the Nav?

- links around the website

3. Can the Nav go in the header?

- Yes, it can be moved in or below the header

4. Does it have to go in the header?

Not necessarily, however, it is prefered to be placed somewhere users are familiar with.

5. Does the Nav contain links to another website?

- No, only links to navigate around the website

6. What goes in Main?

- In the main is where we find the main contents of the page

7. What goes in the Footer?

- Information about the creator of the page, copyright data, sitemap, and links to related websites.

8. HTML Validation

- Use html validator, W3C has its own validator service.
