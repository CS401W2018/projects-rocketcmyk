Questions

1.What is normal flow in HTML?

Normal flow is the default way elements are presented in an html file.

2.What does float do? Why do you need to clear the float?

A float is when an element is not following the normal flow which means it can be positioned in either the left or right side of the layout. It needs to be cleared so that succeeding elements can return to normal flow.

3.What do absolute, fixed, and relative do?

Absolute elements are not included in the flow within an html. Meanwhile fixed elements do not move even when scrolling. and example of this are headers. On the other hand, a relative element can be positioned using the top, right, left, bottom properties.

4. How can you make nice navigation buttons for mobile devices?

You can make nice navigation buttons by additing properties such as display,padding, borders, and colors.
