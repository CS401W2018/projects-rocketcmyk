Questions

1. Why don't we use the border attribute?

- Because the border attribute is considered deprecated which means its no longer used. If ran through a validator it will show an error.

2. What is the easiest way to color every other row a different color?

- The easiest way to do it is to add a class or ID to an element, or you can select the entire specific element you want to color and edit its background color in CSS. This can also be done by adding a pseudoclass that selects odd or even rows.
