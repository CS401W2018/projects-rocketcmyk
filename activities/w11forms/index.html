Questions

1. The form tag has an "action" attribute. What does it do?

- The action attribute tells the browser where to send the data the user inputs.

2. The form tag has a "method" attribute. What does it do?

- The method attribute tells which http method to send the data to the server.

3. What is the difference between a "POST" and a "GET"?

POST is when the data the that is input is not shown in the url, it is generally considered a more secure way when creating forms for users. Meanwhile GET is inputs the users data on the URL as parameters, because it is visible, it is best to use for non-sensitive data.
