1. List out the different ways to select an item.

Tag Name, Descendent Rule, ID, class (multiple tags with the same class. A tag can have more than one class "adjective")

2. What is the difference between an ID and a Class?

- Although both are used to style elements within an HTML, an ID must be uniqu and should only be used on one element with a specific ID. Meanwhile, a class can be used on multiple elements on a page.

3. What is the difference between a SPAN tag and a DIV tag?

- The div tag is used to create a style for a separate section or block within the page. Meanwhile the span tag can be used when you want to style a specific part or portion of a text with the use of inline elements.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A DIV tag is used as a container for structure and grouping.

A header represents the introduction of the page usually containing the company logo and title

The main tag contains the primary contents of the page.

The footer contains the copyright information, contact details, sitemap, and other links to pages

5. What are the different ways to select a color?

Some of the ways to select a color are the folowing:

Pre defined named colors, HTML recognizes a set of pre defined color names but only for simple colors

Hexadecimal Color Codes, using six digits that represents a color, more range of colors can be used

RGB Values, it provides the same range of colors to hex code values except it uses a function where each value should contain a value from 0 to 255

HSL Values, its stands for Hue Saturation,a and Lightness. By using this it is easier to manipulate colors and create harmoniuos color palettes.

Color Picker, by using a color picker, it is easier to select a color as a visual color picker will be provided, the value of the color is then turned into a hex code.
