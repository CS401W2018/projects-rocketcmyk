Questions

1.When should you use a background image, and when should you use the IMG tag?

- The background Image can be used to decorate the body of the website. It is usually used for styling purposes and creating a specific visual aesthetic. The background image does not necessarily need to convey a message as long as it is accessible to the viewers of the webpage. Meanwhile the img tag can be used to include a message to the users and to display important information that can help the users understand the purpose of the page.
