Questions

1. Why is the viewport meta tag important?

- It is used to make sure that websites will be displayed correctly on different devices with different sceen sizes.

2. How can you adjust the CSS for printing?

- You can adjust the css for printing by creating a new css file specifically for making changes for the printable page. This can be done by linking a new css and adding the "media" property inside the link element. In addition to that, you can also change the font to make it easier to read when printed. Simplifying the webpage by removing some links can also help save ink.

3. When would you want to adjust the CSS for printing?

- When you would want to ensure it gets printed correctly into a physical format.
