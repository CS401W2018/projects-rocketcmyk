Questions

1. Why is the viewport meta tag important?

- It is used to make sure that websites will be displayed correctly on different devices with different sceen sizes.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

- The one that takes priority is the one that is written lower.

3. How do you change the CSS for a web page displayed on a phone?

- You can change it for mobile phones by adding a new CSS property which changes the design of the page when it becomes a certain specific screen size that is specified.
