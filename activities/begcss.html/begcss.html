Questions:

1. Explain the difference between inline, embedded, and external CSS.

Inline CSS is directly applied on the html elements using the style attribute. It only styles the specific element it is applied to. On the other hand, embedded CSS affects all elements within the HTML file. Embedded CSS styles are placed within the style tag in the head tag. Meanwhile, external CSS is used when styling an HTML using a separate css file. This can be done by linking the HTML file to the CSS file

2. Explain why you should only use external CSS.

With the use of external CSS, maintaining, organize, and optimizing a page becomes easier. Through an external CSS, all styles can be found in one place making it more convenient. It is also easy to maintain a more consistent design all across the page.

3. What is a selector used for?

A selector in CSS is used to select a sepcific HTML element that you want to style. Selectors can be applied on different element types, specific class, or an element with a unique ID.
