  1. Internet

    2. The Internet is a place where we can access a worldwide system of interconnected networks. These networks allow us to access a variety of data and media through different devices.

  2. Web

    3. It is a system of documents and resources that can be accessed through the internet. It is what we use when we want to browse online, stream videos, play online games, and use online applications.

  3. How are the internet and Web different?

    4. The internet is the infrastructure that allows us to connect to a network. Meanwhile, the internet utilizes this infrastructure so we can use the applications and services to deliver and access content.

  4. ASCII and Unicode

    5. It is an encoding standard that uses a 7-bit code to represent 128 characters primarily in the English letters including numbers and symbols. On the other hand, Unicode is newer and more comprehensive as it can support encoding a wide range of characters from different languages.

  5. What is UTF-8

    6. UTF-8 or Unicode Transformation Format 8 is a character encoding system that uses 1 to 4 bytes. Its space efficiency using only a few bytes allows it to be commonly used in many digital assets such as softwares, websites, and social media.

  6. URL

    7. URL also known as Uniform Resource Locator is an address that is unique to every web page, or a specific resource like an image or document. This also allows us to make navigation around the web a lot easier with just a click.

  7. Domain Name

    8. The Domain name is similar to a URL that points to an address of a specific website except it is easier to remember. It is usually seen on top of the page that comes after www. or https://

  8. HTTP

    9. HTTP or Hypertext Transfer Protocol is a set of communication rules used for transporting information between a client and a web server. Through this we are able to access information and interact with websites.

  9. HTTPS

    10. HTTPS or Hypertect Transfer Protocol Secure is similar to HTTP but with extra security. It encrypts all the communication between the browser and the website protecting sensitive information while browsing.

  10. IP Address

    11. An IP address is a unique address for each device that is connected to the internet. It helps identify your device and allows it to send and receive data across the internet.

  11. DNS

    12. DNS or the Domain Name System functions like the phonebook of the internet. It looks up the domain name that you are looking for and translates it into machine readable IP addresses that can connect you to the website.

  12. W3C Recommendations

    13. W3C Recommendations provide a standard for all developers on how to build websites that can work properly on all devices. They make sure that websites are consistent and easy to use for everyone.

  13. XML, XHTLML, HTML, HTML 5

    14. XML is a flexible language for creating your own custom tags, while XHTML is a more rigid version of HTML that used XML rules. On the other hand HTML is the language used in building websites that uses tags to organize content. Today we have HTML 5 which is the latest version of HTML that can support modern web technologies which is why it is preferred by most developers today.
